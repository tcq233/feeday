### 纯 HTML 最快：Meta Refresh 标签
如果由于某种原因（比如有些平台屏蔽了 JS 脚本）只能使用纯 HTML，可以使用 <meta> 标签。
同样放在 <head> 尽量靠前的位置。将 content 设置为 0 代表 0 秒后立即跳转。

```
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="refresh" content="0;url=https://www.example.com">
    <title>页面跳转中...</title>
</head>
<body>
</body>
</html>
```
